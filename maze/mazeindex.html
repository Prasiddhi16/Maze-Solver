Maze Solver

Draw walls, set start and end points, then watch the algorithm find the path!

Drawing Tools

Algorithm

Breadth-First Search

BFS explores all neighbors at the current depth before moving to the next level. It uses a queue (FIFO).

How to Use

  1. Select a drawing tool (Wall, Erase, Start, or End) from the control panel
  2. Click and drag on the grid to draw walls or place start/end points
  3. Choose your algorithm: DFS (Depth-First Search) or BFS (Breadth-First Search)
  4. Click "Solve Maze" to watch the algorithm find the shortest path
  5. Use "Clear Path" to remove the solution or "Clear All" to start over
